# Laravue Docs

Source code for [Laravue](https://laravue.dev).
Documentation: https://doc.laravue.dev.

## Install vuepress
```
# install
yarn global add vuepress # OR npm install -g vuepress
```

## How to run

```
yarn run dev
```
