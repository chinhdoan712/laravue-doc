# Laravue Docs

Source code for the [Laravue documentation](https://doc.laravue.dev).
Demo site: https://laravue.dev.
Repository: https://github.com/tuandm/laravue

## Install vuepress
```
# install
yarn global add vuepress # OR npm install -g vuepress
```

## How to run

```
yarn run dev
```
